# InterHouse Milestone Rules

## Current milestone
InterHouse MVP+ Lobby & Analytics Polish

## Milestone goal
Make InterHouse feel more complete, legible, and launchable by improving lobby clarity, analytics visibility, and verification discipline.

## Work that belongs in this milestone
- lobby clarity improvements
- analytics/dashboard work
- navigation clarity
- lightweight quality-of-life improvements
- QA passes on recent shipped work
- launch-readiness framing

## Work that does NOT belong unless explicitly approved
- large architecture rewrites
- major gameplay redesign
- deep Solana expansion
- speculative growth features
- broad visual redesign without clear need

## Done-for-now standard
This milestone is in good shape when:
- lobby key stats are clearer
- analytics page exists and is verified
- recent user-facing changes have QA coverage
- there is a launch-readiness checklist
- the next backlog after this milestone is obvious
